# R2 Digital ESLint config

## Whats included?

### extends
- Standard config base;
- Airbnb base;
- Prettier;

### plugins
- Standard;
- Import;
- Prettier;

## Setup

### Node.js

Install dependencies (`pnpm`):
```
pnpm add -D @r2-digital/eslint-config
```

Inside `.eslintrc.json` or `.eslintrc`
```
{
  "extends": "@r2-digital/eslint-config"
}
```
